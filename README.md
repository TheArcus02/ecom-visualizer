# Main purpose

Enable a user to select a small set of fashion items and generate a single AI “Outfit Preview” image that visually combines those items, then provide a clear path to purchase the selected items.

# Pages

- Products: simple grid of items with add/remove to Outfit List.
- Outfit List: lightweight panel/page showing selected items (2–6), reorder and remove.
- Generate Preview: modal/page to trigger AI generation from current Outfit List and optional style preset.
- Preview Display: shows the generated outfit image with actions to regenerate, swap items, and “Buy Look.”
